<span data-ttu-id="1d48d-119">メンバーシップサービスとロールサービスを使用する開発者にとっての一般的な課題は、運用環境でのユーザーとロールの管理です。</span><span class="sxs-lookup"><span data-stu-id="1d48d-119">A common challenge for developers using the Membership and Roles services is managing the users and roles on the production environment.</span></span> <span data-ttu-id="1d48d-120">運用 web サイトからユーザーアカウントを削除したり、新しいロールを追加したり、既存のユーザーを既存のロールに追加したりするにはどうすればよいですか。</span><span class="sxs-lookup"><span data-stu-id="1d48d-120">How do you delete a user account from the production website, add a new role, or add an existing user to an existing role?</span></span> <span data-ttu-id="1d48d-121">このチュートリアルでは、運用 web サイトでユーザーとロールを管理するためのさまざまな手法について説明します。</span><span class="sxs-lookup"><span data-stu-id="1d48d-121">This tutorial explores different techniques for managing users and roles on the production website.</span></span>
