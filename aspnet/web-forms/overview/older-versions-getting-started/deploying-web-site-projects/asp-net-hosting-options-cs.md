2. <span data-ttu-id="739ad-136">ASP.NET のページ、コードファイル、`Bin` フォルダー内のアセンブリ、および CSS や JavaScript ファイルなどの HTML 関連のサポートファイルを同期します。</span><span class="sxs-lookup"><span data-stu-id="739ad-136">Synchronize the ASP.NET pages, code files, the assemblies in the `Bin` folder, and HTML-related support files like CSS and JavaScript files.</span></span>
